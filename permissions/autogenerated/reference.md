## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-request-photos-auth`
- `allow-get-photos-auth-status`
- `allow-request-albums`
- `allow-request-album-medias`
- `allow-request-medias-by-ids`
- `allow-check-album-can-operation`
- `allow-create-album`
- `allow-create-photos`
- `allow-create-videos`
- `allow-delete-album-medias`
- `allow-delete-album`
- `allow-remove-album-medias`
